# Тестовое задание на вакансию «Программист (Qt / C++ / Linux)»

В этом репозитории выполнено три тестовых задачи. 
Текст задания находится в корне репозитория. Код каждой задачи находится в соответствующей папке: `task1`, `task2`, `task3`. 

Сборка для задач 1 и 2 выполняется через Makefile. Для справки используйте

```bash
make help 
```

В корне репозитория лежит файл `.clang-format`, который используется для стандартизации стиля исходного кода. 

## Задачи 1 и 2

Для компиляции используйте:

```bash
make all
```

Все исполняемые файлы будут лежать в папке `build`.

## Задача 3

### Компиляция

1. Перейдите в папку `task3`:

```bash
cd task3
```

2. Сгенерируйте Makefile с помощью `qmake`:

```bash
qmake
```

3. Скомпилируйте проект:

```bash
make
```

### Запуск:

1. В одном терминале выполните:

```bash
./build/task3
```

2. В другом терминале выполните:

```bash
make task2
./build/task2
```

## Очистка проекта:

Чтобы удалить все скомпилированные файлы и вернуть репозиторий к исходному состоянию:

```bash
make clean
```
